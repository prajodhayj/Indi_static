# Indi AI Studio — Premium Video Production Company

Indi AI Studio is a high-end, modern static website designed for a premium video production agency. The site combines traditional filmmaking artistry with cutting-edge AI technology to showcase cinematic excellence.

![Indi AI Studio Preview](images/portfolio-corporate.png)

## ✨ Features

- **Cinematic Hero Section**: Features an immersive background video with a glassmorphism overlay.
- **Modern Design System**: Built with a sleek dark theme, vibrant gradients (Cyan to Purple), and Outfit/Inter typography.
- **Dynamic Stats**: Showcases key performance indicators like projects delivered and industry awards.
- **Full Service Showcase**: Detailed sections for Corporate Films, TV Commercials, Music Videos, and AI Post-Production.
- **Interactive Portfolio**: A grid-based showcase of featured productions with hover effects.
- **Streamlined Process**: Visualizes the production workflow from Discovery to Delivery.
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Contact Integration**: Professional contact form and location details for lead generation.
- **Smooth Animations**: Powered by custom CSS reveals and interactive transitions.

## 🚀 Tech Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Custom design system using CSS Variables, Flexbox, and Grid. No bulky frameworks.
- **JavaScript (ES6)**: Lightweight logic for navigation, scroll reveals, and form handling.
- **FontAwesome**: For premium iconography.
- **Google Fonts**: Utilizing 'Outfit' and 'Inter'.

## 📁 Project Structure

```text
indiai_static/
├── images/          # Assets and images
├── index.html       # Main landing page
├── style.css        # Custom design system and animations
├── script.js        # Interactive components logic
└── logo.png         # Studio branding
```

## 🛠️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/indiai-studio.git
   ```
2. Open `index.html` in any modern web browser.
3. No build step required – it's pure static excellence.
